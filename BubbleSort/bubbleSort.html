1. What could this site help me?

I am a website created by Nguyen Tien Linh from CICT, CTU. I am here to help you to learn about the Bubble Sort algorithm. I have some functions that you may like:

  • Enter an integer or an integer string.
  • Adjust the speed of algorithm implementation.
  • Perform step-by-step of the algorithm.
  • The data table records the arrangement process.

2. How can I add a number?

You can add a number for arrangement by many ways:

  • The first way that you can add a number is enter a number then press the "Add" button on the below section:
  • An other way, you can enter a number string then press the "Add" button on the below section. However, make sure your string does not violate the following rules:
    • It is not empty.
    • It consists of numbers with no more than 30
    • The numbers are separated by the character "," and end with a number.
    • If the number of elements participating in the arrangement is too much, I will only take it until 10.
  • The final way is to import a file. Let the control bar then press the "Enter" button. Now, you should choose an input file containing a string. Similar to the second way, it must satisfy the above constraints.

3. Can I adjust the speed of algorithm?
Yes, you can. Let's see the "Speed" section with three buttons "0.5x", "1x", and "2x". Press one of them to adjust the speed of the implementation.

4. What are the yellow buttons?
They are the buttons to control the implementation algorithm. I will complaint about them:
  • Sort - when click on the "Sort" button, then the algorithm will start.
  • Clear - it removes all the bubbles on the screen.
  • Random - it is similar to the "Clear" button then it recreates new bubbles.
  • Pause - of course, it pauses the implementation and becomes the "Continue" button. You just press it to continue.
  • Import - as you know, it imports a file from your device.

5. How can I view the data table?

Just click on this item then a data table will apprear. It consists of the array after each sorting.
